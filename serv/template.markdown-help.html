Keyboard Shortcuts

(On macOS, use ⌘ instead of CTRL)
- CTRL-x cut
- CTRL-c copy
- CTRL-v paste
- CTRL-a select all
- CTRL-z undo
- CTRL-SHIFT-z redo
- CTRL-b bold the selection
- CTRL-i italics the selection
- CTRL-u underline the selection
- CTRL-l add a hyperlink, change selection into a link
- CTRL-dash toggle bullet(s) for the current line or selected lines 
- TAB indent a line or selected lines
- SHIFT-TAB un-indent a line or selected lines
